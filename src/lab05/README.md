# Лабораторная работа 5: Функции как аргументы. Стратегии и делегаты.

**Предметная область:** Медицина

## Цель работы

Освоение передачи функций как аргументов, применение `map`, `filter`, `sorted`, реализация паттерна «Стратегия».

## Реализованные функции и стратегии

### Функции сортировки
- `by_name` - сортировка по имени
- `by_age` - сортировка по возрасту

### Функции фильтрации
- `is_active` - только активные пациенты
- `has_diagnosis(diagnosis)` - фабрика фильтров по диагнозу

### Callable-объект (паттерн Стратегия)
- `StatusUpdateStrategy` - стратегия обновления статуса пациента

## Методы коллекции

| Метод | Описание |
|-------|----------|
| `sort_by(key_func, reverse=False)` | Сортировка с переданной функцией |
| `filter_by(predicate)` | Фильтрация с переданным предикатом |
| `apply(func)` | Применение функции ко всем элементам |
